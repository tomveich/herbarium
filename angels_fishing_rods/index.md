# Mečík močiarny
- Lat.: Gladiolus palustris Gaudin
- En.: Angels fishing rods  

![Mečík močiarny](./angels_fishing_rods.jpg "Mečík močiarny")

- Európsky druh
- 30-60cm vysoká bylina
- Zákonom chránená kriticky ohrozená rastlina
- Biotiop - mokrade, lesné lúky
