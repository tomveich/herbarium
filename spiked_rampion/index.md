# Zerva klasnatá
- Lat.: Phyteuma spicatum
- En.: Spiked Rampion

![Zerva klasnatá](./spiked_rampion.jpg "Zerva klasnatá")

Čeľaď: Zvončekovité (Campanulaceae)

- Trváca 15-80cm bylina s prízemnou ružicou
- Rastie priamo hore bez vetvenia
