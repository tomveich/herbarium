# Pochybok biely
- Lat.:
- En.: Milkwhite rock jasmine

!["Pochybok biely"](./rock_jasmine.jpg "Pochybok biely")

Čeľaď: Primulaceae

- 3-15cm trváca bylina
- Rastie v horách strednej a juhovýchodnej Európy
- Obľubuje vápencové skaly
