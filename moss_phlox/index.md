# Flox šidlolistý
- Lat.: Phlox subulata
- En.: Moss Phlox

![Flox šidlolistý](./moss_phlox.jpg "Flox šidlolistý")


- Okrasná rastlina
- Zvykne rásť cca 10cm do výšky
- Tvorí husté porasty pri zemi
- Rôzne odrody majú rôzne farby kvetov
- Údajne vôňou pripomína kanabis
