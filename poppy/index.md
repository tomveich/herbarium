# Mak siaty
- Lat.: Papaver somniferum
- En.: Opium poppy

![Mak siaty](./poppy.jpg "Mak siaty")

Čeľaď: Makovité (Papaveraceae)

- Jednoročná bylina
- Pochádza z východnej Indie a Turecka
- Suché semená obsahujú 12x viac vápnika ako mlieko
