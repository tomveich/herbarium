# Horčinka obyčajná
- Lat.: Polygala vulgaris
- En.: Common milkwort

![Horčinka obyčajná](./miklwort.jpg "Horčinka obyčajná")

Čeľaď: Polygalaceae

- Výška: 30cm
- Nízka až plazivá bylina
- Fialové kvety v zhlukoch
