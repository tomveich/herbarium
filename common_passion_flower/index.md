# Múčenka
- Lat.: Passiflora caerulea
- En.: Common passion flower

![Múčenka](./ "Múčenka")

Čeľaď: Passifloraceae

- Rýchlorastúca popínavá rastlina
- Modro-biele voňavé kvety
- 3cm dlhé žlto-orandžové bobule
