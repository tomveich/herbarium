# Mliečnik chvojkový
- Lat.: Euphorbia cyparissias
- En.: Cypress sprunge

![Mliečnik chvojkový](./cypress_sprunge.jpg "Mliečnik chvojkový")

Čeľaď: Mliečnikovité (Euphorbiaceae)

- 20-40cm trváca bylina
- Drevnatá stonka a podzemok
- Plody explodujú až 5m do prostredia
