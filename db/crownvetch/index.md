# Ranostaj Pestrý
Lat.: Securigera varia
En.: Crownvetch

![Ranostaj Pestrý](./crownvetch.jpg "Ranostaj Pestrý")

Čeľaď: Bôbovité (Fabaceae)

- Rastie veľmi rýchlo
- Stala sa inváznou rastlinou
- Je toxická pre kone
